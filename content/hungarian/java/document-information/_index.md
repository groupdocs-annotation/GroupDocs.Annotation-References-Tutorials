# Dokumentuminformációk – Oktatóanyagok a GroupDocs.Annotation Java-hoz

Dokumentuminformációs oktatóanyagaink bemutatják, hogyan kinyerheti és elemezheti a dokumentum részleteit a GroupDocs.Annotation for Java segítségével. Ezek az átfogó útmutatók a fájlinformációk, a támogatott formátumok, az oldalméretek, a szöveges tartalom és a verzióinformációk programozott lekérését tárgyalják. Minden oktatóanyag részletes Java kódpéldákat tartalmaz a dokumentum metaadatainak eléréséhez, amelyek a dokumentum jellemzőin alapuló intelligens feldolgozással javíthatják az annotációs munkafolyamatokat.

### [Támogatott fájlformátumok lekérése a GroupDocs.Annotation for Java fájlban: Átfogó útmutató](./groupdocs-annotation-java-supported-formats/)
