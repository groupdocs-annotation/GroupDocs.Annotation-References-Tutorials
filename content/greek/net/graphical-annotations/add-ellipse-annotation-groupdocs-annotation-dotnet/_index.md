# Πώς να προσθέσετε σχολιασμούς σε PDF χρησιμοποιώντας το GroupDocs.Annotation .NET API

## Εισαγωγή

Βελτιώστε τα έγγραφα PDF σας με διαδραστικές σχολιασμούς, όπως αποσιωπητικά, χρησιμοποιώντας το GroupDocs.Annotation .NET API. Είτε επισημαίνετε βασικές ενότητες είτε παρέχετε οπτικές ενδείξεις, η προσθήκη σχολίων αποσιωπητικού μπορεί να κάνει τα έγγραφά σας πιο ενημερωτικά και ελκυστικά.

**Τι θα μάθετε:**
- Ρύθμιση του GroupDocs.Annotation για .NET
- Δημιουργία και προσαρμογή μιας σχολίασης με αποσιώπηση
- Προσθήκη σχολίων σε συγκεκριμένες σελίδες στο PDF σας
- Αποθήκευση του σχολιασμένου εγγράφου

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία προσθήκης μιας σχολίασης με αποσιώπηση. Βεβαιωθείτε ότι πληροίτε όλες τις προϋποθέσεις πριν ξεκινήσετε.

## Προαπαιτούμενα

Για να παρακολουθήσετε αυτό το σεμινάριο, βεβαιωθείτε ότι έχετε:

## Ρύθμιση του GroupDocs.Annotation για .NET

### Εγκατάσταση

Ξεκινήστε εγκαθιστώντας τη βιβλιοθήκη GroupDocs.Annotation:

**Κονσόλα διαχείρισης πακέτων NuGet:**
```bash
Install-Package GroupDocs.Annotation -Version 25.4.0
```

**.NET CLI:**
```bash
dotnet add package GroupDocs.Annotation --version 25.4.0
```

### Απόκτηση Άδειας

Για να χρησιμοποιήσετε το GroupDocs.Annotation, μπορείτε να κάνετε τα εξής:
- Εγγραφείτε για μια δωρεάν δοκιμαστική περίοδο για να δοκιμάσετε τη βιβλιοθήκη.
- Υποβάλετε αίτηση για προσωρινή άδεια χρήσης για να εξερευνήσετε όλες τις λειτουργίες χωρίς περιορισμούς.
- Αγοράστε μια άδεια χρήσης για μακροπρόθεσμα έργα.

Για ρύθμιση και αρχικοποίηση:
```csharp
using GroupDocs.Annotation;

// Αρχικοποιήστε τον σχολιαστή με τη διαδρομή του εγγράφου PDF σας
Annotator annotator = new Annotator("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

## Οδηγός Εφαρμογής

### Προσθήκη σχολίου έλλειψης σε ένα έγγραφο PDF

Σε αυτήν την ενότητα, θα δούμε πώς να δημιουργούμε και να προσαρμόζουμε μια σχολίαση σε σχήμα έλλειψης.

#### Βήμα 1: Αρχικοποίηση του αντικειμένου σχολιαστή

Ξεκινήστε αρχικοποιώντας το `Annotator` αντικείμενο με τη διαδρομή του εγγράφου PDF σας:
```csharp
using (Annotator annotator = new Annotator(inputPdfPath))
{
    // Θα προσθέσουμε σχολιασμούς σε αυτό το μπλοκ.
}
```

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων σχολιασμού ελλειπτικού

Δημιουργήστε μια παρουσία του `EllipseAnnotation` με τις επιθυμητές ιδιότητες:
```csharp
EllipseAnnotation ellipse = new EllipseAnnotation
{
    BackgroundColor = 65535, // Κίτρινο χρώμα σε μορφή ARGB
    Box = new Rectangle(100, 100, 200, 150), // Θέση (x, y) και μέγεθος (πλάτος, ύψος)
    CreatedOn = DateTime.Now,
    Message = "This is an ellipse annotation",
    Opacity = 0.7f,
    PageNumber = 1, // Ο αριθμός σελίδας για την προσθήκη της σχολίασης
    PenColor = 65535,
    PenStyle = PenStyle.Dot,
    PenWidth = 3,
    Replies = new List<Reply>
    {
        new Reply { Comment = "First comment", RepliedOn = DateTime.Now },
        new Reply { Comment = "Second comment", RepliedOn = DateTime.Now }
    }
};
```

#### Βήμα 3: Προσθήκη της σχολίασης έλλειψης

Προσθέστε την διαμορφωμένη σχολίαση με αποσιώπηση στο έγγραφό σας:
```csharp
annotator.Add(ellipse);
```

#### Βήμα 4: Αποθήκευση του σχολιασμένου εγγράφου

Αποθηκεύστε το σχολιασμένο PDF σε μια καθορισμένη διαδρομή εξόδου:
```csharp
annotator.Save(outputPath);
```
