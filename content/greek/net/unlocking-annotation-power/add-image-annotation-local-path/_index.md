# Προσθήκη σχολίου εικόνας σε έγγραφο (τοπική διαδρομή)

## Εισαγωγή
Το GroupDocs.Annotation για .NET είναι μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να προσθέτουν διάφορους τύπους σχολιασμών σε έγγραφα μέσω προγραμματισμού. Σε αυτό το σεμινάριο, θα μάθουμε πώς να προσθέτουμε σχολιασμούς εικόνας σε ένα έγγραφο χρησιμοποιώντας μια τοπική διαδρομή.
## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:
1. Βασική γνώση της γλώσσας προγραμματισμού C#.
2. Το Visual Studio είναι εγκατεστημένο στο σύστημά σας.
3. GroupDocs.Annotation για εγκατεστημένη βιβλιοθήκη .NET ή tutorialsd στο έργο σας.
4. Ένα έγγραφο εισόδου (π.χ., PDF) και ένα αρχείο εικόνας για σχολιασμό.
## Εισαγωγή χώρων ονομάτων
Αρχικά, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο αρχείο C# σας. Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στις κλάσεις και τις μεθόδους που απαιτούνται για την εργασία με το GroupDocs.Annotation.
```csharp
using System;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
```

## Βήμα 1: Ορισμός διαδρομής εξόδου
Ορίστε τη διαδρομή εξόδου όπου θα αποθηκευτεί το σχολιασμένο έγγραφο.
```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```
## Βήμα 2: Αρχικοποίηση σχολιαστή
Αρχικοποιήστε τον σχολιαστή παρέχοντας τη διαδρομή προς το έγγραφο εισόδου.
```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
    // Ο κώδικας σχολιασμού μπαίνει εδώ
}
```
## Βήμα 3: Δημιουργία σχολιασμού εικόνας
Δημιουργήστε μια παρουσία του `ImageAnnotation` κλάση και καθορίστε τις ιδιότητές της, όπως θέση, αδιαφάνεια, αριθμό σελίδας και διαδρομή εικόνας.
```csharp
ImageAnnotation image = new ImageAnnotation
{
    Box = new Rectangle(100, 100, 100, 100),
    CreatedOn = DateTime.Now,
    Opacity = 0.7,
    PageNumber = 0,
    ImagePath = "image.png"
};
```
## Βήμα 4: Προσθήκη σχολίου
Προσθέστε την σχολίαση της δημιουργημένης εικόνας στο έγγραφο χρησιμοποιώντας το `Add` η μέθοδος του σχολιαστή.
```csharp
annotator.Add(image);
```
## Βήμα 5: Αποθήκευση εγγράφου
Αποθηκεύστε το σχολιασμένο έγγραφο στη διαδρομή εξόδου.
```csharp
annotator.Save(outputPath);
```
## Βήμα 6: Εμφάνιση διαδρομής εξόδου
Εμφανίστε ένα μήνυμα που υποδεικνύει την επιτυχή αποθήκευση του εγγράφου και τη θέση του αρχείου εξόδου.
```csharp
Console.WriteLine($"\nDocument saved successfully.\nCheck output in {outputPath}.");
```

## Σύναψη
Σε αυτό το σεμινάριο, μάθαμε πώς να προσθέτουμε σχολιασμούς εικόνας σε ένα έγγραφο χρησιμοποιώντας το GroupDocs.Annotation για .NET. Ακολουθώντας αυτά τα απλά βήματα, μπορείτε να βελτιώσετε τις δυνατότητες επεξεργασίας εγγράφων σας με ισχυρές λειτουργίες σχολιασμού.
## Συχνές ερωτήσεις
### Μπορώ να προσθέσω σχόλια σε έγγραφα εκτός από PDF;
Ναι, το GroupDocs.Annotation υποστηρίζει διάφορες μορφές εγγράφων, όπως Word, Excel, PowerPoint και άλλες.
### Είναι το GroupDocs.Annotation συμβατό με το .NET Core;
Ναι, το GroupDocs.Annotation είναι συμβατό τόσο με το .NET Framework όσο και με το .NET Core.
### Μπορώ να προσαρμόσω την εμφάνιση των σχολιασμών;
Απολύτως, μπορείτε να προσαρμόσετε την εμφάνιση, το μέγεθος, το χρώμα και άλλες ιδιότητες των σχολιασμών σύμφωνα με τις απαιτήσεις σας.
### Παρέχει το GroupDocs.Annotation υποστήριξη για συνεργατική σχολιασμό;
Ναι, το GroupDocs.Annotation προσφέρει λειτουργίες συνεργατικής σχολιασμού που επιτρέπουν σε πολλούς χρήστες να προσθέτουν σχολιασμούς σε έγγραφα ταυτόχρονα.
### Πού μπορώ να βρω επιπλέον βοήθεια ή υποστήριξη;
Μπορείτε να επισκεφθείτε το φόρουμ GroupDocs.Annotation για βοήθεια και υποστήριξη από την κοινότητα.
