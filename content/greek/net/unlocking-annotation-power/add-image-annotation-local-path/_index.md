## Εισαγωγή
Το GroupDocs.Annotation για .NET είναι μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να προσθέτουν διάφορους τύπους σχολιασμών σε έγγραφα μέσω προγραμματισμού. Σε αυτό το σεμινάριο, θα μάθουμε πώς να προσθέτουμε σχολιασμούς εικόνας σε ένα έγγραφο χρησιμοποιώντας μια τοπική διαδρομή.
## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:
1. Βασικές γνώσεις γλώσσας προγραμματισμού C#.
2. Το Visual Studio είναι εγκατεστημένο στο σύστημά σας.
3. Το GroupDocs.Annotation για τη βιβλιοθήκη .NET έχει εγκατασταθεί ή αναφέρεται στο έργο σας.
4. Ένα έγγραφο εισόδου (π.χ. PDF) και ένα αρχείο εικόνας για σχολιασμό.
## Εισαγωγή χώρων ονομάτων
Αρχικά, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο αρχείο C#. Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στις κλάσεις και τις μεθόδους που απαιτούνται για την εργασία με το GroupDocs.Annotation.
```csharp
using System;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
```

## Βήμα 1: Καθορίστε τη διαδρομή εξόδου
Καθορίστε τη διαδρομή εξόδου όπου θα αποθηκευτεί το σχολιασμένο έγγραφο.
```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```
## Βήμα 2: Εκκίνηση του Annotator
Αρχικοποιήστε τον σχολιαστή παρέχοντας τη διαδρομή προς το έγγραφο εισόδου.
```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
    // Ο κώδικας σχολιασμού πηγαίνει εδώ
}
```
## Βήμα 3: Δημιουργία σχολιασμού εικόνας
 Δημιουργήστε ένα παράδειγμα του`ImageAnnotation`κλάση και καθορίστε τις ιδιότητες του, όπως θέση, αδιαφάνεια, αριθμό σελίδας και διαδρομή εικόνας.
```csharp
ImageAnnotation image = new ImageAnnotation
{
    Box = new Rectangle(100, 100, 100, 100),
    CreatedOn = DateTime.Now,
    Opacity = 0.7,
    PageNumber = 0,
    ImagePath = "image.png"
};
```
## Βήμα 4: Προσθήκη σχολιασμού
 Προσθέστε τον σχολιασμό εικόνας που δημιουργήθηκε στο έγγραφο χρησιμοποιώντας το`Add` μέθοδος του σχολιαστή.
```csharp
annotator.Add(image);
```
## Βήμα 5: Αποθήκευση εγγράφου
Αποθηκεύστε το σχολιασμένο έγγραφο στη διαδρομή εξόδου.
```csharp
annotator.Save(outputPath);
```
## Βήμα 6: Εμφάνιση διαδρομής εξόδου
Εμφανίστε ένα μήνυμα που υποδεικνύει την επιτυχή αποθήκευση του εγγράφου και τη θέση του αρχείου εξόδου.
```csharp
Console.WriteLine($"\nDocument saved successfully.\nCheck output in {outputPath}.");
```

## συμπέρασμα
Σε αυτό το σεμινάριο, μάθαμε πώς να προσθέτουμε σχολιασμούς εικόνας σε ένα έγγραφο χρησιμοποιώντας το GroupDocs.Annotation για .NET. Ακολουθώντας αυτά τα απλά βήματα, μπορείτε να βελτιώσετε τις δυνατότητες επεξεργασίας εγγράφων σας με ισχυρές λειτουργίες σχολιασμού.
