# Προσθήκη σχολιασμού τμήματος κειμένου αναζήτησης στο έγγραφο

## Εισαγωγή
Στον τομέα της ανάπτυξης .NET, το GroupDocs.Annotation ξεχωρίζει ως ένα ισχυρό εργαλείο για απρόσκοπτη σχολιασμό εγγράφων. Είτε είστε έμπειρος προγραμματιστής είτε μόλις μπείτε στον κόσμο του .NET, αυτός ο περιεκτικός οδηγός θα σας καθοδηγήσει στα βασικά στοιχεία της χρήσης του GroupDocs.Annotation για .NET, από την εισαγωγή χώρων ονομάτων έως τον έλεγχο των περιπλοκών της προσθήκης σχολιασμών τμημάτων κειμένου αναζήτησης στο έγγραφα.
## Εισαγωγή
Το GroupDocs.Annotation για .NET δίνει τη δυνατότητα στους προγραμματιστές να ενσωματώνουν τις δυνατότητες σχολιασμού εγγράφων στις εφαρμογές τους χωρίς κόπο. Με το διαισθητικό API και τις ισχυρές του δυνατότητες, οι προγραμματιστές μπορούν να σχολιάσουν διάφορες μορφές εγγράφων, όπως PDF, έγγραφα του Microsoft Office, εικόνες και άλλα.
## Προαπαιτούμενα
Προτού βουτήξετε στο GroupDocs.Annotation για .NET, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

## Εισαγωγή χώρων ονομάτων
Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων για πρόσβαση στις τάξεις και τις μεθόδους GroupDocs.Annotation στο έργο σας .NET:
```csharp
using System;
using System.Collections.Generic;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
using GroupDocs.Annotation.Options;
```
## Βήμα 1: Καθορίστε τη διαδρομή εξόδου
Ξεκινήστε ορίζοντας τη διαδρομή εξόδου όπου θα αποθηκευτεί το σχολιασμένο έγγραφο:
```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```
## Βήμα 2: Εκκίνηση του Annotator
 Στη συνέχεια, αρχικοποιήστε μια παρουσία του`Annotator` τάξη παρέχοντας τη διαδρομή προς το έγγραφο που θέλετε να σχολιάσετε:
```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
```
## Βήμα 3: Δημιουργήστε σχολιασμό τμήματος κειμένου αναζήτησης
 Δημιουργώ ένα`SearchTextFragment` αντικείμενο με τις επιθυμητές ιδιότητες, όπως κείμενο προς αναζήτηση, μέγεθος γραμματοσειράς, οικογένεια γραμματοσειρών, χρώμα γραμματοσειράς και χρώμα φόντου:
```csharp
SearchTextFragment searchText = new SearchTextFragment()
{
    Text = "Welcome to GroupDocs",
    FontSize = 10,
    FontFamily = "Calibri",
    FontColor = 65535,
    BackgroundColor = 16761035,
};
```
## Βήμα 4: Προσθήκη σχολιασμού
 Προσθέστε τον σχολιασμό τμήματος κειμένου αναζήτησης που δημιουργήθηκε στο έγγραφο χρησιμοποιώντας το`Add` μέθοδος του σχολιαστή:
```csharp
annotator.Add(searchText);
```
## Βήμα 5: Αποθήκευση σχολιασμένου εγγράφου
Αποθηκεύστε το σχολιασμένο έγγραφο στην καθορισμένη διαδρομή εξόδου:
```csharp
annotator.Save(outputPath);
```
## Βήμα 6: Εμφάνιση μηνύματος επιτυχίας
Ενημερώστε τον χρήστη ότι το έγγραφο αποθηκεύτηκε με επιτυχία:
```csharp
Console.WriteLine($"\nDocument saved successfully.\nCheck output in {outputPath}.");
```

Συμπερασματικά, το GroupDocs.Annotation για .NET απλοποιεί τη διαδικασία προσθήκης σχολιασμών σε έγγραφα, ενισχύοντας τη συνεργασία και τις διαδικασίες ελέγχου εγγράφων. Ακολουθώντας τα βήματα που περιγράφονται σε αυτόν τον οδηγό, μπορείτε να ενσωματώσετε απρόσκοπτα τις δυνατότητες σχολιασμού εγγράφων στις εφαρμογές σας .NET.
