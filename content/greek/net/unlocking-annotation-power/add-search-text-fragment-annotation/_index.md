# Προσθήκη σχολίου τμήματος κειμένου αναζήτησης σε έγγραφο

## Εισαγωγή
Στον τομέα της ανάπτυξης .NET, το GroupDocs.Annotation ξεχωρίζει ως ένα ισχυρό εργαλείο για την απρόσκοπτη προσθήκη σχολίων σε έγγραφα. Είτε είστε έμπειρος προγραμματιστής είτε μόλις μπήκατε στον κόσμο του .NET, αυτό το ολοκληρωμένο σεμινάριο θα σας καθοδηγήσει στα βασικά στοιχεία της χρήσης του GroupDocs.Annotation για .NET, από την εισαγωγή χώρων ονομάτων έως την κατανόηση των περιπλοκών της προσθήκης σχολίων σε τμήματα κειμένου αναζήτησης στα έγγραφά σας.
## Εισαγωγή
Το GroupDocs.Annotation για .NET δίνει τη δυνατότητα στους προγραμματιστές να ενσωματώνουν δυνατότητες σχολιασμού εγγράφων στις εφαρμογές τους χωρίς κόπο. Με το εύχρηστο API και τις ισχυρές λειτουργίες του, οι προγραμματιστές μπορούν να προσθέτουν σχολιασμούς σε διάφορες μορφές εγγράφων, όπως PDF, έγγραφα του Microsoft Office, εικόνες και άλλα.
## Προαπαιτούμενα
Πριν ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Annotation για .NET, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

## Εισαγωγή χώρων ονομάτων
Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων για να αποκτήσετε πρόσβαση στις κλάσεις και τις μεθόδους GroupDocs.Annotation στο έργο .NET σας:
```csharp
using System;
using System.Collections.Generic;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
using GroupDocs.Annotation.Options;
```
## Βήμα 1: Ορισμός διαδρομής εξόδου
Ξεκινήστε ορίζοντας τη διαδρομή εξόδου όπου θα αποθηκευτεί το σχολιασμένο έγγραφο:
```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```
## Βήμα 2: Αρχικοποίηση σχολιαστή
Στη συνέχεια, αρχικοποιήστε μια παρουσία του `Annotator` κλάση παρέχοντας τη διαδρομή προς το έγγραφο που θέλετε να σχολιάσετε:
```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
```
## Βήμα 3: Δημιουργία σχολίου τμήματος κειμένου αναζήτησης
Δημιουργήστε ένα `SearchTextFragment` αντικείμενο με τις επιθυμητές ιδιότητες, όπως κείμενο προς αναζήτηση, μέγεθος γραμματοσειράς, οικογένεια γραμματοσειρών, χρώμα γραμματοσειράς και χρώμα φόντου:
```csharp
SearchTextFragment searchText = new SearchTextFragment()
{
    Text = "Welcome to GroupDocs",
    FontSize = 10,
    FontFamily = "Calibri",
    FontColor = 65535,
    BackgroundColor = 16761035,
};
```
## Βήμα 4: Προσθήκη σχολίου
Προσθέστε την σχολίαση του δημιουργημένου τμήματος κειμένου αναζήτησης στο έγγραφο χρησιμοποιώντας το `Add` μέθοδος του σχολιαστή:
```csharp
annotator.Add(searchText);
```
## Βήμα 5: Αποθήκευση σχολιασμένου εγγράφου
Αποθηκεύστε το σχολιασμένο έγγραφο στην καθορισμένη διαδρομή εξόδου:
```csharp
annotator.Save(outputPath);
```
## Βήμα 6: Εμφάνιση μηνύματος επιτυχίας
Ενημερώστε τον χρήστη ότι το έγγραφο έχει αποθηκευτεί με επιτυχία:
```csharp
Console.WriteLine($"\nDocument saved successfully.\nCheck output in {outputPath}.");
```

## Σύναψη
Συμπερασματικά, το GroupDocs.Annotation για .NET απλοποιεί τη διαδικασία προσθήκης σχολιασμών σε έγγραφα, ενισχύοντας τη συνεργασία και τις διαδικασίες αναθεώρησης εγγράφων. Ακολουθώντας τα βήματα που περιγράφονται σε αυτόν τον οδηγό, μπορείτε να ενσωματώσετε απρόσκοπτα τις δυνατότητες σχολιασμού εγγράφων στις εφαρμογές .NET που διαθέτετε.
