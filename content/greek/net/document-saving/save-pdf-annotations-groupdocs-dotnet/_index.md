Αυτή η ενότητα θα σας καθοδηγήσει στην εφαρμογή της λειτουργίας αποθήκευσης σχολιασμών χρησιμοποιώντας το GroupDocs.Annotation για .NET. Θα την αναλύσουμε σε διαχειρίσιμα βήματα για να διασφαλίσουμε σαφήνεια και ευκολία.

### Άνοιγμα ροής αρχείων
Ξεκινήστε ρυθμίζοντας το περιβάλλον σας με τις απαραίτητες διαδρομές αρχείων:
```csharp
// Ορίστε εδώ τη διαδρομή εισαγωγής PDF για να ορίσετε το αρχείο σας.
string inputPdfPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "input.pdf");

// Ορισμός καταλόγου εξόδου για την αποθήκευση σχολιασμένων εγγράφων
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", 
    "result" + Path.GetExtension(inputPdfPath));
```

### Χρήση του Annotator για την αποθήκευση σχολίων
**Βήμα 1: Ανοίξτε τη Ροή αρχείων**
Ανοίξτε μια ροή αρχείων του εγγράφου εισόδου σας. Αυτό το βήμα είναι κρίσιμο, καθώς προετοιμάζει το έγγραφο για την επεξεργασία σχολίων.
```csharp
using (FileStream fs = new FileStream(inputPdfPath, FileMode.Open))
{
    // Δημιουργήστε μια παρουσία σχολιαστή με τη ροή αρχείων
    using (Annotator annotator = new Annotator(fs))
    {
        // Αποθήκευση σχολίων στο αρχικό έγγραφο
        annotator.Save(outputPath);
    }
}
```
- **Εξήγηση:** Ο `FileStream` Το αντικείμενο χρησιμοποιείται εδώ για να ανοίξετε το έγγραφο PDF σας. Δημιουργώντας μια παρουσία του `Annotator`, ορίζετε ένα πλαίσιο για τις σχολιασμοί σας.
- **Παράμετροι και τιμές επιστροφής:** Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά, καθώς καθορίζουν από πού διαβάζεται το αρχείο εισόδου και από πού αποθηκεύεται το αρχείο εξόδου.
