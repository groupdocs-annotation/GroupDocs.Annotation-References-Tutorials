# Φόρτωση PDF με Προστασία Κωδικού με GroupDocs.Annotation Java – Προηγμένες Λειτουργίες

Έτοιμοι να αξιοποιήσετε πλήρως τις δυνατότητες σχολιασμού εγγράφων στις εφαρμογές Java σας; Έχετε κατακτήσει τα βασικά και τώρα ήρθε η ώρα να **φορτώσετε PDF με προστασία κωδικού** ενώ εκμεταλλεύεστε τις πιο ισχυρές προηγμένες λειτουργίες που προσφέρει το GroupDocs.Annotation για Java. Αυτός ο οδηγός σας δείχνει πώς να διαχειρίζεστε κρυπτογραφημένα έγγραφα, να περιστρέφετε PDF, να προσθέτετε προσαρμοσμένες γραμματοσειρές, να διαχειρίζεστε τη μνήμη αποδοτικά και να εξάγετε μεταδεδομένα—όλα σε έναν συνομιλιακό, βήμα‑βήμα στυλ που κάνει τις σύνθετες έννοιες εύκολες στην κατανόηση.

## Quick Answers
- **Πώς φορτώνω ένα PDF με προστασία κωδικού;** Χρησιμοποιήστε το `AnnotationConfig` για να παρέχετε τον κωδικό κατά το άνοιγμα του εγγράφου.  
- **Μπορώ να περιστρέψω ένα PDF σε Java;** Ναι—το GroupDocs.Annotation παρέχει μια μέθοδο `rotate` που λειτουργεί σε οποιαδήποτε σελίδα. αντικείμενασαρμοσμένες ενός κρυπτογραφημένου αρχείου παρέχοντας τον σωστό κωδικό, ώστε να μπορείτε να το διαβάσετε, να το σχολιάσετε και να το αποθηκεύσετε χωρίς να παραβιάσετε την ασφάλεια. Το GroupDocs.Annotation Java απλοποιεί αυτή τη διαδικασία με ενσωματωμένες παραμέτρους αυθεντικοποίησης.

## Γιατί να χρησιμοποιήσετε Προηγμένες Λειτουργίες όπως Περιστροφή, Προσαρμοσμένες Γραμματοσειρές και Διαχείριση Μνήμης;
- **Επαγγελματική παρουσίαση:** Περιστρέψτε τις σελίδες ώστε να ταιριάζουν με σαρωμένα έγγραφα ή προτιμήσεις χρήστη.  
- **Συνέπεια εμπορικού σήματος:** Εφαρμόστε προσαρμοσμένες γραμματοσειρές ώστε τα σχόλια να ταιριάζουν με το εταιρικό στυλ.  
- **Κλιμακωσιμότητα:** Η αποδοτική διαχείριση μνήμης επιτρέπει στην εφαρμογή σας να επεξεργάζεται εκατοντάδες σελίδες χωρίς εξάντληση πόρων.  
- **Συμμόρφωση:** Η εξαγωγή μεταδεδομένων σας βοηθά να πληροίτε απαιτήσεις ελέγχου και κανονισμών.

## Πώς να Φορτώσετε PDF με Προστασία Κωδικού με GroupDocs.Annotation Java
### Βήμα 1: Διαμορφώστε τη Μηχανή Σχολιασμού με τον Κωδικό του Εγγράφου
Αρχικά, δημιουργήστε ένα αντικείμενο `AnnotationConfig` και ορίστε τον κωδικό. Αυτό ενημερώνει τη βιβλιοθήκη πώς να αποκρυπτογραφήσει το αρχείο πριν ξεκινήσει οποιαδήποτε εργασία σχολιασμού.

### Βήμα 2: Ανοίξτε το Έγγραφο και Επαληθεύστε την Πρόσβαση
Χρησιμοποιήστε το `AnnotationApi` για να φορτώσετε το έγγραφο. Εάν ο κωδικός είναι σωστός, το API επιστρέφει ένα αντικείμενο `Document` με το οποίο μπορείτε να εργαστείτε· διαφορετικά, ρίχνει μια εξαίρεση αυθεντικοποίησης.

### Βήμα 3: Εφαρμόστε Προηγμένες Λειτουργίες (Περιστροφή, Προσαρμοσμένες Γραμματοσειρές, Μεταδεδομένα)
Μόλις ανοίξει το έγγραφο, μπορείτε να:
- **Περιστρέψετε σελίδες** χρησιμοποιώντας `document.rotate(pageNumber, rotationAngle)`.  
- **Προσθέστε προσαρμοσμένες γραμματοσειρές** καταχωρίζοντας το αρχείο γραμματοσειράς με `annotationConfig.addFont(filePath)`.  
- **Εξάγετε μεταδεδομένα** μέσω `document.getDocumentInfo()` για να διαβάσετε τίτλο, συγγραφέα, ημερομηνία δημιουργίας κ.λπ.

### Βήμα 4: Αποθηκεύστε το Σχολιασμένο Έγγραφο με Ασφάλεια
Αφού κάνετε αλλαγές, αποθηκεύστε το έγγραφο με τον ίδιο ή νέο κωδικό για να το διατηρήσετε προστατευμένο.

## Τι Κάνει αυτές τις Λειτουργίες «Προηγμένες»;
Οι προηγμένες λειτουργίες στο GroupDocs.Annotation υπερβαίνουν την απλή επισήμανση κειμένου και τα βασικά σχήματα. Αυτές οι δυνατότητες σας επιτρέπουν να:
- **Προσαρμόστε την οπτική εμπειρία** με προσαρμοσμένες γραμματοσειές και με ελέγχους ποιότητας εικόνας και βελτιώσεις επεξεργασίας  
- **Δημιουργήσετε έξυπνα φίλτρα** για διαχείριση σχολίων μεγάλης κλίμακας  
- **Αντιμετωπίσετε σύνθετες απαιτήσεις ασφαλείας** με επεξεργασία εγγράφων με προστασία κτιωμένη λειτουργικότητα

## Συχνές Ερωτήσεις
**Q: Μπορώ να σχολιάσω ένα PDF που είναι τόσο προστατευμένο με κωδικό όσο και κρυπτογραφημένο με ψηφιακό πιστοποιητικό;**  
A: Ναι. Παρέχετε τον κωδικό (ή τα διαπιστευτήρια του πιστοποιητικού) μέσω `AnnotationConfig` πριν ανοίξετε το έγγραφο· η βιβλιοθήκη θα διαχειριστεί αυτόματα την αποκρυπτογράφηση.

**Q: Πώς μπορώ να περιστρέψω μια συγκεκριμένη σελίδα χωρίς να επηρεάσω το υπόλοιπο του εγγράφου;**  
A: Χρησιμοποιήστε τη μέθοδο `rotate(pageNumber, rotationAngle)` στο αντικείμενο `Document`, καθορίζοντας τη σελίδα-στόχο και τη ζητούμενη γωνία (90°, 180° ή 270°).

**Q: Ποιος είναι ο προτεινόμενος τρόπος για να προσθέσω μια προσαρμοσμένη γραμματοσειρά για το κείμενο των σχολίων;**  
A: Καταχωρίστε το αρχείο γραμματοσειράς με `annotationConfig.addFont("/path/to/font.ttf")` πριν δημιουργήσετε οποιαδήποτε σχολιαστική κειμένου, και στη συνέχεια αναφέρετε το όνομα της γραμματοσειράς στις ρυθμίσεις στυλ του σχολίου.

**Q: Πώς μπορώ να μειώσω τη χρήση μνήμης κατά την επεξεργασία μεγάλων PDF με πολλά σχόλια;**  
A: Ενεργοποιήστε τη lazy loading, απελευθερώστε τα αντικείμενα `Annotation` μετά τη χρήση, και σκεφτείτε την επεξεργασία του εγγράφου σε μικρότερα εύρη σελίδων αντί να φορτώνετε ολόκληρο το αρχείο ταυτόχρονα.

**Q: Είναι δυνατόν να εξάγω μεταδεδομένα PDF όπως συγγραφέας, τίτλος και ημερομηνία δημιουργίας;**  
A: Ναι. Καλέστε `document.getDocumentInfo()` για να λάβετε ένα αντικείμενο `DocumentInfo` που περιέχει τα τυπικά πεδία μεταδεδομένων.
