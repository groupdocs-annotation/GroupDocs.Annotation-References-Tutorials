## Οδηγός Εφαρμογής

### Σχολιασμός διαγραφής κειμένου

Στόχος είναι να δείξουμε πώς να προσθέσετε μια σχολίαση διακριτής διαγραφής κειμένου σε έγγραφα PDF χρησιμοποιώντας το GroupDocs.Annotation.

#### Βήμα 1: Ρύθμιση παραμέτρων διαδρομών εγγράφων

Ορίστε διαδρομές εισόδου και εξόδου για το έγγραφό σας:

```java
String inputFilePath = "path/to/your/document/directory/sample.pdf";
String outputPath = "path/to/your/output/directory/AddTextStrikeoutAnnotation_output.pdf";
```

#### Βήμα 2: Αρχικοποίηση σχολιαστή

Δημιουργήστε μια παρουσία του `Annotator` για να διαχειριστείτε το έγγραφο PDF στο οποίο θέλετε να προσθέσετε σχόλια:

```java
final Annotator annotator = new Annotator(inputFilePath);
```

#### Βήμα 3: Προετοιμασία απαντήσεων (σχόλια)

Προσθέστε σχόλια ή απαντήσεις που σχετίζονται με τους σχολιασμούς σας, εάν χρειάζεται:

```java
Reply reply1 = new Reply();
reply1.setComment("First comment");
reply1.setRepliedOn(Calendar.getInstance().getTime());

List<Reply> replies = new ArrayList<>();
replies.add(reply1);
```

#### Βήμα 4: Ορισμός σημείων σχολιασμού

Καθορίστε συντεταγμένες για την περιοχή διαγραφής στο έγγραφό σας:

```java
Point point1 = new Point(80, 730);
Point point2 = new Point(240, 730);
List<Point> points = Arrays.asList(point1, point2);
```

#### Βήμα 5: Δημιουργία και ρύθμιση παραμέτρων σχολιασμού διαγραφής

Ρύθμιση ενός `StrikeoutAnnotation` αντικείμενο με απαραίτητες ιδιότητες όπως χρώμα γραμματοσειράς, μήνυμα και αδιαφάνεια:

```java
StrikeoutAnnotation strikeout = new StrikeoutAnnotation();
strikeout.setCreatedOn(Calendar.getInstance().getTime());
strikeout.setFontColor(65535); // Κίτρινος
strikeout.setMessage("This is a strikeout annotation");
strikeout.setOpacity(0.7);
strikeout.setPageNumber(0); 
strikeout.setPoints(points);
strikeout.setReplies(replies);
```

#### Βήμα 6: Προσθήκη σχολίου στο έγγραφο

Προσθέστε τη διαμορφωμένη σχολίαση στο έγγραφό σας χρησιμοποιώντας `Annotator`:

```java
annotator.add(strikeout);
```

#### Βήμα 7: Αποθήκευση και απόρριψη

Αποθηκεύστε το σχολιασμένο PDF σας και δημοσιεύστε τους πόρους:

```java
annotator.save(outputPath);
annotator.dispose();
```
