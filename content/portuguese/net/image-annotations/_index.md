# Tutoriais de anotação de imagem para GroupDocs.Annotation .NET

Domine técnicas de anotação de imagens com nossos tutoriais detalhados do GroupDocs.Annotation .NET. Estes guias práticos demonstram como inserir anotações de imagens a partir de caminhos locais ou URLs remotas, posicionar e redimensionar imagens, aplicar configurações de transparência e personalizar as propriedades de anotação de imagens. Cada tutorial inclui exemplos completos de código C# para incorporar elementos visuais que aprimoram a clareza e a comunicação do documento.

## Tutoriais disponíveis

### [Adicionar anotações de imagem a documentos usando GroupDocs.Annotation para .NET](./add-image-annotations-groupdocs-net/)
Aprenda a integrar o GroupDocs.Annotation aos seus projetos .NET para aprimorar documentos com anotações de imagem. Aumente o engajamento do usuário e simplifique a colaboração.

### [Guia completo para adicionar anotações de imagem no .NET com GroupDocs.Annotation](./guide-image-annotations-groupdocs-dotnet/)
Aprenda a adicionar anotações em imagens usando o GroupDocs.Annotation para .NET. Aprimore documentos nos setores de educação, jurídico e saúde.

### [Implementar anotação de imagem em PDFs usando GroupDocs.Annotation .NET e URLs remotos](./groupdocs-annotation-net-image-remote-url/)
Aprenda a adicionar anotações de imagens de URLs remotas em documentos PDF usando o GroupDocs.Annotation para .NET. Aprimore seus fluxos de trabalho com documentos com este guia completo.
