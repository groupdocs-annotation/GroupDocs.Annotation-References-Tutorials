# Tutoriais de informações de documentos para GroupDocs.Annotation .NET

Nossos tutoriais de informações sobre documentos mostram como extrair e analisar detalhes de documentos usando o GroupDocs.Annotation para .NET. Esses guias abrangentes abordam a recuperação programática de informações de arquivos, formatos suportados, dimensões de página, conteúdo de texto e informações de versão. Cada tutorial inclui exemplos detalhados de código C# para acessar metadados de documentos que podem aprimorar seus fluxos de trabalho de anotação com processamento inteligente baseado nas características do documento.
