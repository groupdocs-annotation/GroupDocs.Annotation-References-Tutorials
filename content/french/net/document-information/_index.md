# Tutoriels d'information sur les documents pour GroupDocs.Annotation .NET

Nos tutoriels sur les informations documentaires vous expliquent comment extraire et analyser les détails d'un document à l'aide de GroupDocs.Annotation pour .NET. Ces guides complets abordent la récupération programmatique des informations sur les fichiers, les formats pris en charge, les dimensions des pages, le contenu textuel et les informations de version. Chaque tutoriel inclut des exemples de code C# détaillés pour accéder aux métadonnées des documents et améliorer vos workflows d'annotation grâce à un traitement intelligent basé sur les caractéristiques du document.
