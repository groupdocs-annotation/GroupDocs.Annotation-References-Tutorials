## Questions fréquemment posées

**Q : Comment activer page range saving java dans un service Spring Boot ?**  
A: Injectez le bean `AnnotationApi`, configurez le `SaveOptions` avec la plage de pages souhaitée, et exposez un point d'accès qui déclenche l'opération d'enregistrement de manière asynchrone.

**Q : Puis‑je combiner page range saving avec le versionnage de documents java ?**  
A: Oui – incluez les métadonnées de version dans le nom de fichier (par ex., `Contract_v2_2024-09-15.pdf`) ou stockez‑les dans une propriété personnalisée du document avant l'enregistrement.

**Q : Quels formats offrent une fidélité d'annotation complète ?**  
A: Le PDF est le format le plus fiable pour préserver tous les types d'annotation ; d'autres formats peuvent perdre certaines fonctionnalités interactives.

**Q : Comment puis‑je surveiller l'utilisation de la mémoire lors de gros enregistrements ?**  
A: Utilisez `Runtime.getRuntime().freeMemory()` de Java et consignez la mémoire avant et après l'enregistrement, ou intégrez un outil de profilage comme VisualVM.

**Q : Existe‑t‑il un moyen d'enregistrer en lot plusieurs documents avec des plages de pages différentes ?**  
A: Absolument – parcourez votre collection de documents, définissez des `SaveOptions` individuels pour chacun, et exécutez les enregistrements en parallèle en utilisant le `ExecutorService` de Java.
