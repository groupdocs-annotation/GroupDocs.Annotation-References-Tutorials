# Handledningar för länkannotering för GroupDocs.Annotation .NET

Våra handledningar för länkannoteringar visar hur du förbättrar dokument med interaktiva element med GroupDocs.Annotation för .NET. Dessa omfattande guider visar hur du lägger till hyperlänkar till externa webbplatser, interna dokumentplatser, e-postadresser och andra resurser. Varje handledning innehåller exempel på fungerande C#-kod för att implementera fullt fungerande länkannoteringar med anpassningsbara inställningar för utseende och beteende.
