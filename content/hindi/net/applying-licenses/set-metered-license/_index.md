# मीटरयुक्त लाइसेंस सेट करें

## परिचय
.NET के लिए GroupDocs.Annotation एक शक्तिशाली लाइब्रेरी है जो डेवलपर्स को अपने .NET अनुप्रयोगों में दस्तावेज़ एनोटेशन क्षमताओं को सहजता से जोड़ने का अधिकार देती है। चाहे आप एक दस्तावेज़ प्रबंधन प्रणाली, सहयोग मंच, या कोई एप्लिकेशन बना रहे हों जिसमें दस्तावेज़ समीक्षा और मार्कअप शामिल हो, .NET के लिए GroupDocs.Annotation प्रक्रिया को सुव्यवस्थित करने के लिए उपकरणों का एक व्यापक सेट प्रदान करता है।
इस ट्यूटोरियल में, हम GroupDocs.Annotation .NET के लिए मीटर्ड लाइसेंस स्थापित करने की प्रक्रिया के बारे में विस्तार से जानेंगे। मीटर्ड लाइसेंस आपको केवल आपके द्वारा उपभोग किए गए संसाधनों के लिए भुगतान करने की अनुमति देता है, जिससे यह किसी भी पैमाने की परियोजनाओं के लिए एक लागत प्रभावी समाधान बन जाता है। नीचे दिए गए चरणों का पालन करके, आप संसाधन उपयोग को अनुकूलित करते हुए और बजटीय नियंत्रण बनाए रखते हुए GroupDocs.Annotation को अपने .NET एप्लिकेशन में निर्बाध रूप से एकीकृत करने में सक्षम होंगे।
## आवश्यक शर्तें
ट्यूटोरियल में जाने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित शर्तें हैं:
1.  .NET लाइब्रेरी के लिए GroupDocs.Annotation: लाइब्रेरी को यहां से डाउनलोड करें[वेबसाइट](https://releases.groupdocs.com/annotation/net/).
2. ग्रुपडॉक्स खाते तक पहुंच: मीटर्ड लाइसेंस स्थापित करने के लिए आवश्यक सार्वजनिक और निजी कुंजी प्राप्त करने के लिए आपको ग्रुपडॉक्स खाते की आवश्यकता होगी। यदि आपके पास अभी तक कोई खाता नहीं है, तो आप निःशुल्क परीक्षण के लिए साइन अप कर सकते हैं[यहाँ](https://releases.groupdocs.com/).
3. C# और .NET फ्रेमवर्क की बुनियादी समझ: C# प्रोग्रामिंग भाषा और .NET फ्रेमवर्क से परिचित होना इस ट्यूटोरियल में उल्लिखित चरणों को लागू करने के लिए फायदेमंद होगा।

## नामस्थान आयात करें
आरंभ करने के लिए, अपने C# प्रोजेक्ट में आवश्यक नामस्थान आयात करना सुनिश्चित करें। ये नामस्थान GroupDocs.Annotation कार्यक्षमता के साथ इंटरैक्ट करने के लिए आवश्यक हैं।
```csharp
using System;
```
## चरण 1: सार्वजनिक और निजी कुंजी प्राप्त करें
मीटर्ड लाइसेंस स्थापित करने से पहले, आपको अपने ग्रुपडॉक्स खाता डैशबोर्ड से अपनी सार्वजनिक और निजी कुंजी प्राप्त करनी होगी।
1. अपने ग्रुपडॉक्स खाते में लॉग इन करें।
2. लाइसेंस प्रबंधन अनुभाग पर जाएँ.
3. GroupDocs द्वारा प्रदान की गई अपनी सार्वजनिक और निजी कुंजियों की प्रतिलिपि बनाएँ।
## चरण 2: मीटरयुक्त लाइसेंस सेट करें
एक बार जब आप अपनी सार्वजनिक और निजी कुंजी प्राप्त कर लेते हैं, तो आप अपने .NET एप्लिकेशन में मीटर्ड लाइसेंस सेट कर सकते हैं।
```csharp
string publicKey = "*****"; // ***** को अपनी सार्वजनिक कुंजी से बदलें
string privateKey = "*****"; // ***** को अपनी निजी कुंजी से बदलें
Metered metered = new Metered();
metered.SetMeteredKey(publicKey, privateKey);
Console.WriteLine("License set successfully.");
```

## निष्कर्ष
अंत में, GroupDocs.Annotation .NET के लिए मीटर्ड लाइसेंस स्थापित करना एक सीधी प्रक्रिया है जो आपके दस्तावेज़ एनोटेशन परियोजनाओं के लिए कुशल संसाधन उपयोग और लागत-प्रभावशीलता सुनिश्चित करती है। इस ट्यूटोरियल में बताए गए चरणों का पालन करके, आप GroupDocs.Annotation को अपने .NET एप्लिकेशन में सहजता से एकीकृत कर सकते हैं और दस्तावेज़ सहयोग और समीक्षा क्षमताओं को बढ़ा सकते हैं।
## अक्सर पूछे जाने वाले प्रश्न

हाँ, .NET के लिए GroupDocs.Annotation का उपयोग वाणिज्यिक और गैर-व्यावसायिक दोनों परियोजनाओं में किया जा सकता है। हालाँकि, आपको अपनी परियोजना आवश्यकताओं के आधार पर एक उपयुक्त लाइसेंस प्राप्त करने की आवश्यकता है।
### क्या .NET के लिए GroupDocs.Annotation के लिए कोई परीक्षण संस्करण उपलब्ध है?
 हां, आप यहां जाकर .NET के लिए GroupDocs.Annotation के निःशुल्क परीक्षण का लाभ उठा सकते हैं[इस लिंक](https://releases.groupdocs.com/).

 आप GroupDocs फोरम पर जाकर तकनीकी सहायता प्राप्त कर सकते हैं[यहाँ](https://forum.groupdocs.com/c/annotation/10).
### क्या कोई अस्थायी लाइसेंस विकल्प उपलब्ध हैं?
 हाँ, आप अल्पकालिक उपयोग या मूल्यांकन उद्देश्यों के लिए GroupDocs से एक अस्थायी लाइसेंस प्राप्त कर सकते हैं। मिलने जाना[इस लिंक](https://purchase.groupdocs.com/temporary-license/) अधिक जानकारी के लिए।
### क्या मैं अपनी परियोजना आवश्यकताओं के अनुसार एनोटेशन सुविधाओं को अनुकूलित कर सकता हूँ?
हाँ, .NET के लिए GroupDocs.Annotation व्यापक अनुकूलन विकल्प प्रदान करता है, जिससे आप अपनी विशिष्ट परियोजना आवश्यकताओं के अनुरूप एनोटेशन सुविधाओं को तैयार कर सकते हैं।
