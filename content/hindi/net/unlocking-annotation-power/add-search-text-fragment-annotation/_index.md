# दस्तावेज़ में खोज टेक्स्ट फ़्रैगमेंट एनोटेशन जोड़ें

## परिचय
.NET विकास के क्षेत्र में, GroupDocs.Annotation दस्तावेजों को सहजता से एनोटेट करने के लिए एक शक्तिशाली उपकरण के रूप में सामने आता है। चाहे आप एक अनुभवी डेवलपर हों या .NET की दुनिया में कदम रख रहे हों, यह व्यापक ट्यूटोरियल आपको .NET के लिए GroupDocs.Annotation का उपयोग करने की अनिवार्यताओं से परिचित कराएगा, नामस्थानों को आयात करने से लेकर आपके दस्तावेज़ों में खोज टेक्स्ट फ़्रैगमेंट एनोटेशन जोड़ने की पेचीदगियों में महारत हासिल करने तक।
## परिचय
GroupDocs.Annotation for .NET डेवलपर्स को अपने अनुप्रयोगों में दस्तावेज़ एनोटेशन क्षमताओं को आसानी से शामिल करने की शक्ति देता है। अपने सहज API और मजबूत सुविधाओं के साथ, डेवलपर्स PDF, Microsoft Office दस्तावेज़, चित्र और अधिक सहित विभिन्न दस्तावेज़ स्वरूपों को एनोटेट कर सकते हैं।
## आवश्यक शर्तें
.NET के लिए GroupDocs.Annotation में गोता लगाने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ मौजूद हैं:

## नामस्थान आयात करें
सबसे पहले, अपने .NET प्रोजेक्ट में GroupDocs.Annotation क्लासेस और विधियों तक पहुँचने के लिए आवश्यक नामस्थान आयात करें:
```csharp
using System;
using System.Collections.Generic;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
using GroupDocs.Annotation.Options;
```
## चरण 1: आउटपुट पथ परिभाषित करें
आउटपुट पथ को परिभाषित करके आरंभ करें जहां एनोटेट दस्तावेज़ सहेजा जाएगा:
```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```
## चरण 2: एनोटेटर आरंभ करें
इसके बाद, एक उदाहरण आरंभ करें `Annotator` उस दस्तावेज़ का पथ प्रदान करके क्लास में जाएँ जिस पर आप टिप्पणी करना चाहते हैं:
```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
```
## चरण 3: खोज टेक्स्ट फ़्रैगमेंट एनोटेशन बनाएँ
एक बनाने के `SearchTextFragment` वांछित गुणों वाला ऑब्जेक्ट, जैसे खोजने के लिए पाठ, फ़ॉन्ट आकार, फ़ॉन्ट परिवार, फ़ॉन्ट रंग और पृष्ठभूमि रंग:
```csharp
SearchTextFragment searchText = new SearchTextFragment()
{
    Text = "Welcome to GroupDocs",
    FontSize = 10,
    FontFamily = "Calibri",
    FontColor = 65535,
    BackgroundColor = 16761035,
};
```
## चरण 4: एनोटेशन जोड़ें
बनाए गए खोज पाठ खंड एनोटेशन को दस्तावेज़ में जोड़ें `Add` व्याख्याकार की विधि:
```csharp
annotator.Add(searchText);
```
## चरण 5: एनोटेट दस्तावेज़ सहेजें
एनोटेट किए गए दस्तावेज़ को निर्दिष्ट आउटपुट पथ पर सहेजें:
```csharp
annotator.Save(outputPath);
```
## चरण 6: सफलता संदेश प्रदर्शित करें
उपयोगकर्ता को सूचित करें कि दस्तावेज़ सफलतापूर्वक सहेजा गया है:
```csharp
Console.WriteLine($"\nDocument saved successfully.\nCheck output in {outputPath}.");
```

## निष्कर्ष
निष्कर्ष में, GroupDocs.Annotation for .NET दस्तावेज़ों में एनोटेशन जोड़ने, सहयोग बढ़ाने और दस्तावेज़ समीक्षा प्रक्रियाओं की प्रक्रिया को सरल बनाता है। इस गाइड में बताए गए चरणों का पालन करके, आप अपने .NET अनुप्रयोगों में दस्तावेज़ एनोटेशन क्षमताओं को सहजता से एकीकृत कर सकते हैं।
## अक्सर पूछे जाने वाले प्रश्न
### क्या GroupDocs.Annotation सभी दस्तावेज़ प्रारूपों के साथ संगत है?
हां, GroupDocs.Annotation पीडीएफ, माइक्रोसॉफ्ट ऑफिस दस्तावेज़, चित्र आदि सहित दस्तावेज़ प्रारूपों की एक विस्तृत श्रृंखला का समर्थन करता है।
### क्या मैं एनोटेशन के स्वरूप को अनुकूलित कर सकता हूँ?
बिल्कुल! GroupDocs.Annotation एनोटेशन के लिए व्यापक अनुकूलन विकल्प प्रदान करता है, जिससे आप फ़ॉन्ट आकार, रंग और शैली जैसे गुणों को समायोजित कर सकते हैं।
### क्या GroupDocs.Annotation के लिए कोई निःशुल्क परीक्षण उपलब्ध है?
हां, आप खरीदारी करने से पहले इसकी सुविधाओं और क्षमताओं का पता लगाने के लिए GroupDocs.Annotation के निःशुल्क परीक्षण का उपयोग कर सकते हैं [यहाँ](https://releases.groupdocs.com/)..
### मैं GroupDocs.Annotation के लिए समर्थन कहां पा सकता हूं?
ग्रुपडॉक्स.एनोटेशन के साथ समर्थन और सहायता के लिए, आप ग्रुपडॉक्स पर जा सकते हैं [मंच](https://forum.groupdocs.com/c/annotation/10) एनोटेशन से संबंधित प्रश्नों और चर्चाओं के लिए समर्पित।
### मैं GroupDocs.Annotation के लिए अस्थायी लाइसेंस कैसे प्राप्त करूं?
आप GroupDocs के माध्यम से GroupDocs.Annotation के लिए एक अस्थायी लाइसेंस प्राप्त कर सकते हैं [वेबसाइट](https://purchase.groupdocs.com/temporary-license/), जिससे आप उत्पाद का पूर्ण मूल्यांकन कर सकेंगे।
