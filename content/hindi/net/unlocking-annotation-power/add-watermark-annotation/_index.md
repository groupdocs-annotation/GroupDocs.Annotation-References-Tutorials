## परिचय
इस ट्यूटोरियल में, हम .NET के लिए GroupDocs.Annotation का उपयोग करके दस्तावेज़ में वॉटरमार्क एनोटेशन जोड़ने की प्रक्रिया के बारे में जानेंगे। वॉटरमार्क एनोटेशन किसी दस्तावेज़ की स्थिति को इंगित करने, उसे गोपनीय के रूप में चिह्नित करने या कोई अन्य प्रासंगिक जानकारी जोड़ने के लिए उपयोगी होते हैं।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1.  .NET के लिए GroupDocs.Annotation: आप इसे यहां से डाउनलोड कर सकते हैं[यहाँ](https://releases.groupdocs.com/annotation/net/).
2. विजुअल स्टूडियो: सुनिश्चित करें कि आपके सिस्टम पर विजुअल स्टूडियो स्थापित है।
3. C# का बुनियादी ज्ञान: कोड उदाहरणों को समझने और लागू करने के लिए C# प्रोग्रामिंग भाषा से परिचित होना आवश्यक है।

## नामस्थान आयात करें

इससे पहले कि हम कोडिंग शुरू करें, आइए आवश्यक नामस्थान आयात करें:

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using GroupDocs.Annotation.Models;
using GroupDocs.Annotation.Models.AnnotationModels;
```

अब, आइए वॉटरमार्क एनोटेशन जोड़ने की प्रक्रिया को कई चरणों में विभाजित करें:

## चरण 1: आउटपुट पथ को परिभाषित करें

 सबसे पहले, हमें आउटपुट पथ को परिभाषित करने की आवश्यकता है जहां एनोटेटेड दस्तावेज़ सहेजा जाएगा। हम उपयोग करेंगे`Path` कक्षा से`System.IO` फ़ाइल नाम के साथ आउटपुट निर्देशिका पथ को संयोजित करने के लिए नेमस्पेस।

```csharp
string outputPath = Path.Combine("Your Document Directory", "result" + Path.GetExtension("input.pdf"));
```

## चरण 2: एनोटेटर प्रारंभ करें

इसके बाद, हम इनपुट दस्तावेज़ का पथ प्रदान करके एनोटेटर को आरंभ करेंगे। यह हमें दस्तावेज़ में एनोटेशन जोड़ने की अनुमति देगा।

```csharp
using (Annotator annotator = new Annotator("input.pdf"))
{
    // एनोटेशन कोड यहां जाएगा
}
```

## चरण 3: वॉटरमार्क एनोटेशन बनाएं

अब, आइए वांछित गुणों जैसे कोण, स्थिति, पाठ, फ़ॉन्ट रंग, अस्पष्टता इत्यादि के साथ एक वॉटरमार्क एनोटेशन ऑब्जेक्ट बनाएं।

```csharp
WatermarkAnnotation watermark = new WatermarkAnnotation
{
    Angle = 75,
    Box = new Rectangle(200, 200, 100, 50),
    CreatedOn = DateTime.Now,
    Text = "Watermark",
    FontColor = 65535,
    FontSize = 12,
    Message = "This is watermark annotation",
    Opacity = 0.7,
    PageNumber = 0,
    AutoScale = true,
    HorizontalAlignment = HorizontalAlignment.Center,
    VerticalAlignment = VerticalAlignment.Center,
    Replies = new List<Reply>
    {
        new Reply
        {
            Comment = "First comment",
            RepliedOn = DateTime.Now
        },
        new Reply
        {
            Comment = "Second comment",
            RepliedOn = DateTime.Now
        }
    }
};
```

## चरण 4: वॉटरमार्क एनोटेशन जोड़ें

 अब, हम इसका उपयोग करके दस्तावेज़ में वॉटरमार्क एनोटेशन जोड़ देंगे`Add` एनोटेटर ऑब्जेक्ट की विधि।

```csharp
annotator.Add(watermark);
```

## चरण 5: दस्तावेज़ सहेजें

अंत में, हम एनोटेटेड दस्तावेज़ को निर्दिष्ट आउटपुट पथ पर सहेजेंगे।

```csharp
annotator.Save(outputPath);
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए GroupDocs.Annotation का उपयोग करके किसी दस्तावेज़ में वॉटरमार्क एनोटेशन कैसे जोड़ा जाए। दस्तावेज़ों को प्रासंगिक जानकारी के साथ चिह्नित करने या उनकी स्थिति दर्शाने के लिए वॉटरमार्क एनोटेशन एक मूल्यवान उपकरण है।

## अक्सर पूछे जाने वाले प्रश्न

### प्रश्न: क्या मैं वॉटरमार्क एनोटेशन के स्वरूप को अनुकूलित कर सकता हूँ?

उत्तर: हां, आप अपनी आवश्यकताओं के अनुसार वॉटरमार्क को तैयार करने के लिए विभिन्न गुणों जैसे टेक्स्ट, फ़ॉन्ट आकार, रंग, अस्पष्टता, स्थिति इत्यादि को अनुकूलित कर सकते हैं।

### प्रश्न: क्या .NET के लिए GroupDocs.Annotation विभिन्न दस्तावेज़ प्रारूपों के साथ संगत है?

उ: हां, GroupDocs.Annotation पीडीएफ, माइक्रोसॉफ्ट वर्ड, एक्सेल, पावरपॉइंट और छवि प्रारूपों सहित दस्तावेज़ प्रारूपों की एक विस्तृत श्रृंखला का समर्थन करता है।

### प्रश्न: क्या मैं एक ही दस्तावेज़ में एकाधिक एनोटेशन जोड़ सकता हूँ?

उ: बिल्कुल, GroupDocs.Annotation आपको व्यापक दस्तावेज़ मार्कअप को सक्षम करते हुए, एक ही दस्तावेज़ में विभिन्न प्रकार के कई एनोटेशन जोड़ने की अनुमति देता है।

### प्रश्न: क्या GroupDocs.Annotation सहयोगी एनोटेशन के लिए सहायता प्रदान करता है?

उत्तर: हाँ, GroupDocs.Annotation उपयोगकर्ताओं को टिप्पणियाँ, उत्तर और एनोटेशन जोड़ने की अनुमति देकर सहयोगात्मक एनोटेशन की सुविधा प्रदान करता है, जिससे टीम के सदस्यों के बीच प्रभावी सहयोग को बढ़ावा मिलता है।

### प्रश्न: क्या .NET के लिए GroupDocs.Annotation का कोई परीक्षण संस्करण उपलब्ध है?

 उत्तर: हां, आप यहां से नि:शुल्क परीक्षण संस्करण डाउनलोड कर सकते हैं[यहाँ](https://releases.groupdocs.com/).
