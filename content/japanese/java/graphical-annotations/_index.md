## GroupDocs.Annotation を使用した Java PDF ウォーターマークの適用方法

**java pdf watermark** を追加するのは、`WatermarkAnnotation` オブジェクトを作成し、プロパティを設定して文書を保存するだけで簡単です。この方法はテキストと画像の両方のウォーターマークに対応し、不透明度、回転、位置をカスタマイズできます。以下は概要フローです（詳細なコードは後述の専用ウォーターマークガイドにあります）。

1. `AnnotationApi` で **PDF をロード** します。  
2. **WatermarkAnnotation を作成**し、`Text` または `Image` タイプを選択します。  
3. **外観を設定** – 色、不透明度、回転、スケール。  
4. **アノテーションを**対象ページに **添付** します。  
5. **更新された PDF を**ストレージに **保存** します。  

プログラムでウォーターマークを適用することで、手作業なしでエクスポートされるすべての PDF に同じ視覚的保護が付与されます。

## 一般的な問題のトラブルシューティング

### アノテーションが表示されない

**問題**：アノテーションはレンダリングされるが、最終文書に表示されない。  
**解決策**：z‑インデックスとレイヤー順序を確認し、アノテーションが文書コンテンツの背後に配置されていないことを確認します。

### 大規模文書でのパフォーマンス問題

**問題**：多数のアノテーションを追加するとレンダリングが遅くなる、またはメモリ問題が発生する。  
**解決策**：アノテーションにページングを実装し、アノテーションプーリングを使用し、描画操作を最適化します。

### スタイルの不一致

**問題**：PDFビューアによってアノテーションの見た目が異なる。  
**解決策**：標準的なアノテーションプロパティを使用し、複数のビューアでテストします。高度なスタイルはすべてのビューアでサポートされない場合があります。

### 座標系の問題

**問題**：アノテーションが誤った位置に表示される。  
**解決策**：座標系の理解を確認します。GroupDocs は他のライブラリとは異なる座標原点を使用しています。

## よくある質問

**Q: 同じ PDF にテキストと画像の両方のウォーターマークを追加できますか？**  
A: はい。GroupDocs.Annotation を使用すると、テキスト用と画像用に別々の `WatermarkAnnotation` オブジェクトを作成し、任意のページに配置できます。

**Q: 印刷後もウォーターマークが見えるようにするには？**  
A: 不透明度を 0.3〜0.5 の範囲に設定し、コントラストのある色を選択します。ほとんどの PDF ビューアは印刷時にアノテーションレイヤーを保持します。

**Q: ウォーターマークを特定のページだけに適用できますか？**  
A: もちろん可能です。ウォーターマークアノテーション作成時に対象ページ番号を指定できます。

**Q: ウォーターマークは PDF のファイルサイズに大きく影響しますか？**  
A: シンプルなテキストウォーターマークはサイズへの影響がほとんどありません。画像ウォーターマークは画像解像度に比例してサイズが増加するため、大規模文書では圧縮画像を使用してください。

**Q: ウォーターマーク機能に別途ライセンスは必要ですか？**  
A: いいえ。ウォーターマークアノテーションは標準の GroupDocs.Annotation for Java ライセンスに含まれています。
