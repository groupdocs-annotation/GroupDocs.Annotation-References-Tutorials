## 常见问题

**Q: 我可以在 Web API 中使用 GroupDocs.Annotation 创建可填写的 PDF 表单吗？**  
A: 可以——该库在 ASP.NET Core、MVC 和 Web API 项目中同样适用。只需加载 PDF，添加表单字段注释，然后将结果流式返回给客户端。

**Q: 我如何从扫描的 PDF 中提取元数据？**  
A: 使用 Document Information API 读取内置元数据。对于扫描的 PDF，可能需要先运行 OCR；该库可与 GroupDocs.Parser 集成进行文本提取。

**Q: 能否为受密码保护的 PDF 生成预览图像？**  
A: 完全可以。打开文档时提供密码，然后调用预览方法即可在不泄露内容的情况下渲染缩略图。

**Q: 推荐的将公司徽标作为图像印章插入的方式是什么？**  
A: 使用 Image Annotations 教程——将徽标加载为流，设置所需的位置和大小，然后将其添加为印章注释。

**Q: 如何批量处理数千个文档进行注释？**  
A: 利用 Annotation Management 的批量操作，并在并行循环或 Azure Function 中运行，以在保持低内存使用的同时最大化吞吐量。
