# Hướng dẫn thông tin tài liệu cho GroupDocs.Annotation Java

Các hướng dẫn toàn diện này bao gồm việc truy xuất thông tin tệp, định dạng được hỗ trợ, kích thước trang, nội dung văn bản và thông tin phiên bản theo chương trình. Mỗi hướng dẫn đều bao gồm các ví dụ mã Java chi tiết để truy cập siêu dữ liệu tài liệu có thể nâng cao quy trình chú thích của bạn bằng cách xử lý thông minh dựa trên các đặc điểm của tài liệu.
