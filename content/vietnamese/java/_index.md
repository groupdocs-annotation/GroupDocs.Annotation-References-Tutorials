# GroupDocs.Annotation for Java - Hướng dẫn API chú thích tài liệu

GroupDocs.Annotation for Java là một API chú thích tài liệu mạnh mẽ cho phép bạn **annotate PDF Java** tài liệu cũng như Word, nhúng khả năng chú thích trực tiếp vào các ứng dụng Java của bạn, bạn có thể xây dựng công tác tài liệu phong
- **Bạn có thể chú thích gì?** PDFs, DOCX, XLSX, PPTX, HTML, hình ảnh, và hơn nữa.  
- **Bạn có cần công cụ bên ngoài không?** Không—mọi thứ chạy từ một JAR duy nhất.  
- **Các loại chú thích nào dấu văn bản, gạch chân, gạch ngang, hình dạng liên kết và trường biểu mẫu.

### Add annotations Java – Đánh dấu đồ họa cốt lõi
Từ các đánh dấu đơn giản đến các hình dạng phức tạp, thư viện cung cấp mô hình thống nhất cho mọi loại chú thích, giúp dễ dàng chuyển đổi giữa chú thích văn bản và đồ họa trong cùng một tài liệu.
