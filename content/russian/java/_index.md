## Как аннотировать PDF с помощью Java и GroupDocs.Annotation?

Класс `AnnotationApi` предоставляет методы, такие как `load` для открытия документа и `save` для записи аннотированного файла.  
`HighlightAnnotation` представляет собой разметку выделения текста с настраиваемым цветом и непрозрачностью.  
`TextAnnotation` добавляет окно комментария, привязанное к месту в документе.

Загрузите целевой PDF с помощью `AnnotationApi.load("sample.pdf")`, создайте нужные объекты аннотаций (например, `HighlightAnnotation`, `TextAnnotation`), задайте свойства, такие как цвет, автор и позиция, затем вызовите `AnnotationApi.save("output.pdf")`. Этот сквозной процесс позволяет вам **create pdf annotations** всего в несколько строк Java‑кода, эффективно обрабатывая большие файлы.
