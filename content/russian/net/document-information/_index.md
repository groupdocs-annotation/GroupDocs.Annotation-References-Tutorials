# Учебники по информации о документе для GroupDocs.Annotation .NET

Наши руководства по информации о документах покажут вам, как извлекать и анализировать сведения о документах с помощью GroupDocs.Annotation для .NET. Эти всеобъемлющие руководства охватывают извлечение информации о файлах, поддерживаемых форматах, размерах страниц, текстовом содержимом и информации о версии программным способом. Каждое руководство включает подробные примеры кода C# для доступа к метаданным документа, которые могут улучшить ваши рабочие процессы аннотирования с помощью интеллектуальной обработки на основе характеристик документа.
