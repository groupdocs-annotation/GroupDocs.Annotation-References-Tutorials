# Учебники по аннотациям ссылок для GroupDocs.Annotation .NET

Наши руководства по аннотациям ссылок покажут вам, как улучшить документы с помощью интерактивных элементов с помощью GroupDocs.Annotation для .NET. Эти всеобъемлющие руководства показывают, как добавлять гиперссылки на внешние веб-сайты, внутренние расположения документов, адреса электронной почты и другие ресурсы. Каждое руководство включает рабочие примеры кода C# для реализации полностью функциональных аннотаций ссылок с настраиваемыми параметрами внешнего вида и поведения.
