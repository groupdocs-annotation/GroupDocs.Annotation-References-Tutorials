## Domande Frequenti

**D: Posso usare GroupDocs.Annotation per creare moduli PDF compilabili in una Web API?**  
R: Sì – la libreria funziona altrettanto bene in progetti ASP.NET Core, MVC e Web API. Basta caricare il PDF, aggiungere le annotazioni di tipo campo modulo e inviare lo stream risultante al client.

**D: Come estraggo i metadati da un PDF scansionato?**  
R: Usa le API Document Information per leggere i metadati incorporati. Per PDF scansionati, potrebbe essere necessario eseguire prima l'OCR; la libreria si integra con GroupDocs.Parser per l'estrazione del testo.

**D: È possibile generare immagini di anteprima per PDF protetti da password?**  
R: Assolutamente. Fornisci la password al momento dell'apertura del documento, quindi chiama i metodi di preview per renderizzare le miniature senza esporre il contenuto.

**D: Qual è il modo consigliato per inserire il logo aziendale come timbro immagine?**  
R: Segui il tutorial Image Annotations – carica il logo come stream, imposta posizione e dimensione desiderate, e aggiungilo come annotazione timbro.

**D: Come posso processare in batch migliaia di documenti per l'annotazione?**  
R: Sfrutta le operazioni batch di Annotation Management e eseguili all'interno di un ciclo parallelo o di una Azure Function per massimizzare il throughput mantenendo basso l'uso di memoria.
